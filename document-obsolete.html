int Document::insertImage(int row, int column, const QImage & image, double xOffset, double yOffset, double xScale = 1, double yScale = 1)

This is an overloaded function.

Insert an image to current active worksheet to the position row, column with the given xOffset, yOffset, xScale and yScale.

void Document::setCurrentWorksheet(int index)

Set current worksheet to be the sheet at index.

See also currentWorksheet() and selectWorksheet().

void Document::setCurrentWorksheet(const QString & name)

Set current selected worksheet to be the sheet named name.

See also selectWorksheet().

bool Document::setSheetName(const QString & name)

Rename current worksheet to new name. Returns true if the name defined successful.

See also renameWorksheet().
