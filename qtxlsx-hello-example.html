Xlsx Hello Example

This is a simplest xlsx examples.

This example demonstrates how to generate a simplest .xlsx file with Qt Xlsx Library.

Create an object of the class QXlsx::Document.

QXlsx::Document xlsx;

Set the cells of worksheet.

xlsx.write("A1", "Hello Qt!");
xlsx.write("A2", 12345);
xlsx.write("A3", "=44+33");
xlsx.write("A4", true);
xlsx.write("A5", "http://qt-project.org");

Save it.

xlsx.save();
